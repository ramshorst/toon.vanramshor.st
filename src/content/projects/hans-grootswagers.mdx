## Context

Hans Grootswagers is a prominent Dutch artist known mainly for bronze sculptures. The goal of the project was to create a website that could present his work, explain his artistic process, and support an international audience in multiple languages.

## What I built

I built the site in PHP and MySQL using my own CMS. The website was available in four languages and included editable sections for sculptures, paintings, galleries, events, and documentation of the casting process behind the bronze works. All of it could be maintained through an admin portal.

## The technical challenge

The main challenge was structuring a multilingual content platform around several different content types: artworks, exhibitions, galleries, and process pages. The CMS needed to make all of that manageable for non-technical editing while keeping the structure consistent across four languages.

## Outcome

The site proved to be durable over a very long lifetime. It only needed a single major technical intervention later on for a PHP migration, which made it a good example of building a maintainable CMS-driven website with a long shelf life.
