## Context

Conclu was an early project to recreate the experience of a French online classifieds platform. The goal was to build the whole product from scratch and learn backend development the hard way: no framework, direct PHP, direct database design, and all the core mechanics behind a listings marketplace.

## What I built

I designed and developed the platform myself, including listing creation, listing pages, database structure, relational logic, and email flows. It covered the essential features you would expect from a classifieds website: posting ads, browsing listings, and handling the supporting backend infrastructure.

## The technical challenge

The real challenge was learning how dynamic web applications worked at a low level. Building it without a framework meant figuring out routing, templating, forms, persistence, and database relationships manually. It became a hands-on way to understand how PHP applications and MySQL schemas fit together in a real product.

## Outcome

This project was my entry point into programming. It taught me the fundamentals of backend web development, relational databases, and product architecture by building a complete marketplace-style platform from the ground up.
