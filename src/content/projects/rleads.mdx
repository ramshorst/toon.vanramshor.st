## Context

r/leads is a side project built with a co-founder around a simple insight: Reddit is full of people actively looking for solutions, but finding those conversations manually is slow, inconsistent, and doesn't scale. The product automates that — monitoring subreddits for genuine buyer intent signals and surfacing opportunities the moment they appear.

<Screenshot src="/rleads-hero.png" alt="r/leads landing page" display="wide" href="https://rleads.ai/" />

## What we built

The stack is React and TypeScript on the front end. The core matching engine uses vector embeddings to identify relevant opportunities — posts are embedded and compared against a product description vector, giving semantic matching that goes well beyond keyword search. This lets the product detect genuine buying intent and pain points rather than just literal mentions.

The back end is deployed on Railway, with Cloudfront in front for distribution. When a match is found, the user gets an alert within 15 minutes via Slack, Discord, or email, along with an AI-suggested response tailored to the thread.

<Screenshot src="/rleads-app.png" alt="Opportunity feed with AI-suggested response" display="wide" href="https://rleads.ai/" />

## How we built it

A meaningful part of the development was done using Claude Code in yolo mode — letting the agent run autonomously across the codebase, each session isolated in its own Docker container. It was a genuine test of AI-assisted development at speed, and it worked.
